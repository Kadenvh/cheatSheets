Tokenization Sandbox

See how text is chopped into subword tokens.

Step 1: Turn text into numbers

Real LLMs use a trained tokenizer (often based on byte-pair encoding). Here we simulate something similar by breaking text into common subword pieces like trans, form, and er. Each colored chip stands for a token ID the model can understand.

Attention Heatmap

Click a word to see which other words it "looks at".

Step 2: Contextual weighting

The small neural network quietly learns powerful patterns.

In a transformer, every token computes "how much to care" about every other token. We show a toy attention matrix: darker cells mean stronger attention. Try clicking different words and see how the pattern shifts.

Positional Encoding Waves

Visualize sine/cosine patterns over token positions.

Step 3: Order without recurrence

Transformers add a deterministic pattern to each position so the model can distinguish "first", "middle", and "last" tokens. We show a toy version: each column is a position; each row is a sine wave with a different frequency.
