Concept: Data Exploration with Pandas
Detail: Quick reference for pandas data exploration. Import: `import pandas as pd`. Load: `pd.read_csv('file.csv')`. View: `.head(n=5)`. Types: `.dtypes`. Counts: `.value_counts()`. Stats: `.describe()`. DataFrame is 2D table, Series is 1D column. Access single column: `df['col']` (Series), multiple: `df[['a','b']]` (DataFrame). Normalize counts: `.value_counts(normalize=True)`.
