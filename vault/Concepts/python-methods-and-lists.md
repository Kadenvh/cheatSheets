Concept: Python Methods & Lists
Detail: String methods: `.lower()` converts to lowercase. List methods: `.sort()` sorts ascending in-place, `.append(item)` adds to end. Parameter `reverse=True` for descending sort. Methods use parentheses (actions), attributes don't (stored info). List manipulation: `my_list.sort(reverse=True)`. String: `category.lower()`.
