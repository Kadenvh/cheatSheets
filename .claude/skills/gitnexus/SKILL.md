# GitNexus Skills

Parent directory for GitNexus sub-skills. Each subdirectory contains its own SKILL.md.

This is a bundle parent. Invoke the specific sub-skill matching your task:

- `gitnexus-exploring/` — Architecture and code flow exploration
- `gitnexus-guide/` — GitNexus usage reference
- `gitnexus-impact-analysis/` — Change safety and dependency analysis
